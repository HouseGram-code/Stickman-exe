## üá∫üá∏ English

**Stickman.exe** is a 2.5D psychological horror game inspired by classic creepypastas like *Sonic.exe*. You play as a simple stickman wandering through a seemingly peaceful world. But the further you go, the more the world corrupts. Something is watching you. Something is following you.

### ‚ú® Features
- **2.5D Perspective:** Pseudo-3D environment built entirely with CSS 3D transforms.
- **Dynamic Atmosphere:** The world changes as you progress (from a sunny day to a burning nightmare).
- **Cross-Platform:** Playable on PC (Keyboard) and Mobile (On-screen touch controls).
- **Jumpscares & Audio:** Creepy sound effects and sudden scares using the Web Audio API.
- **Character Select Screen:** Dynamic character status based on your survival.
